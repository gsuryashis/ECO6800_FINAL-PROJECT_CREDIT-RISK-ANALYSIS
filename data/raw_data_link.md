# Raw Data

The project data files are too large for version control and are hosted externally.

## Access

- **Google Drive folder:** <https://drive.google.com/drive/u/2/folders/1V1iKS0rcghr6K5paWJDK38X3K8lmDn5P>
- **Original source:** [LendingClub Issued Loans – Kaggle](https://www.kaggle.com/datasets/husainsb/lendingclub-issued-loans?resource=download)

## Setup instructions

1. Download `lc_loan.csv` from the Drive folder or Kaggle link above.
2. Place the file at `data/raw/lc_loan.csv` before running any pipeline scripts.
